About Platform

About Question Hub

Question Hub is a modern, high-performance academic resource directory designed specifically for the students of H.H. THE RAJAH'S College. It centralizes previous semesters' model and university examination papers across all major departments, allowing students to access, prepare, and excel in their studies.

Core Features

🔍

Smart Search & Instant Filtering

Filter the entire database dynamically by subject names, course codes, departments, semesters, and years without refreshing the page.

📅

Interactive Study Planner

Register a student account to set study milestones, compile customized checklists for specific subjects, and track preparation statuses.

🏆

Gamified Badges & Contributions

Upload verified question papers to earn recognition points, rank up on the global Leaderboard, and unlock achievement badges for helping peers.

👁

Dual-Screen Optimized Previews

Read PDFs directly with an intelligent reader which automatically optimizes rendering for both desktop and mobile web configurations.

Guidelines

All contributions must be clear, complete PDF files with visible course details. Uploaded papers go through administrator review before appearing on the main listings page to maintain repository quality.
